不要なサービスの停止

不要なサービスの停止

利用しないサービスが起動している事によるセキュリティホール化や、
意図しないサービスによるリソース消費を抑えるために不要なサービスは停止しておく

現在起動しているサービスの確認 

# systemctl list-units --type service |grep active
